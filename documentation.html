Documentation

Anvil is a high-performance C++ library for solving ordinary differential equations (ODEs). It is designed for scientific computing, numerical simulation, and engineering systems.

Overview

The library solves systems of ordinary differential equations by numerically integrating their evolution over time.

General form:

$$ \frac{dy}{dt} = f(t, y) $$

where \( y \in \mathbb{R}^n \) and \( f \) defines the system dynamics.

Mathematical background

Numerical integration approximates continuous systems using discrete steps:

$$ y_{n+1} = y_n + h \, \Phi(t_n, y_n, h) $$

where:

  • \( h \): step size
  • \( \Phi \): numerical integration function

Accuracy depends on local truncation error and global error accumulation.

Core API

anvil::solve

High-level solver entry point.

auto result = anvil::solve(
    system,
    y0,
    t0,
    t1,
    anvil::RK4{}
);

anvil::Vector

Represents system state.

anvil::Vector y0 = {
    1.0,
    0.0,
    0.5
};

System types

Non-stiff systems

Well-behaved systems suitable for explicit methods.

Stiff systems

Require adaptive or implicit solvers for stability.

Autonomous systems

Systems where dynamics do not depend explicitly on time.

Solvers

RK4 (Runge-Kutta 4)

Fixed-step explicit solver for smooth systems. 

anvil::RK4 solver;

RK45 (Adaptive)

Adaptive solver with embedded error estimation. 

anvil::RK45 solver;

Step control

Step size determines the trade-off between accuracy and performance:

  • Small step → higher accuracy
  • Large step → faster execution

Error control

Adaptive solvers use tolerance parameters: 

anvil::RK45 solver;

solver.set_rtol(1e-6);
solver.set_atol(1e-9);

Example

Exponential decay system:

#include <anvil/anvil.hpp>

int main() {

    auto system = [](
        double t,
        const anvil::Vector& y
    ) {
        return anvil::Vector{
            -y[0]
        };
    };

    anvil::Vector y0 = {1.0};

    auto result = anvil::solve(
        system,
        y0,
        0.0,
        5.0,
        anvil::RK4{}
    );

    return 0;
}

Performance notes

  • SIMD optimizations enabled in release builds
  • RK4 is faster for fixed-step simulations
  • RK45 is more accurate for dynamic systems

Limitations

  • No PDE support
  • No symbolic solving
  • Real-valued systems only

Notes

Anvil is optimized for performance-critical scientific workloads.
