Install

Anvil is a high-performance C++ library for solving ordinary differential equations (ODEs). It is designed for numerical simulation, scientific computing, and real-time systems.

The library is header-only, meaning no build step is required beyond including the headers.

Quick start

Include Anvil and create a basic solver instance: 

#include <anvil/anvil.hpp>

int main() {

    anvil::RK4<double> solver;

    solver.set_step_size(0.01);
    solver.set_initial_condition(1.0);

    auto result = solver.solve([](double t, double y) {
        return -y;
    }, 0.0, 10.0);

    return 0;
}

Requirements

  • C++17 or higher
  • GCC 9+, Clang 10+, MSVC 2019+
  • Optional: CMake 3.15+

CMake (recommended)

Using FetchContent:

include(FetchContent)

FetchContent_Declare(
  anvil
  GIT_REPOSITORY https://github.com/anvil/anvil.git
  GIT_TAG v1.0.0
)

FetchContent_MakeAvailable(anvil)

target_link_libraries(app PRIVATE anvil::anvil)

Package managers

vcpkg

vcpkg install anvil

Conan

[requires]
anvil/1.0.0

[generators]
CMakeToolchain

Manual installation

You can also integrate Anvil manually by copying the include/anvil directory into your project. 

git clone https://github.com/anvil/anvil.git

cd anvil

cp -r include/anvil /your/project/include/

Verify installation

Compile the following minimal example: 

#include <iostream>
#include <anvil/anvil.hpp>

int main() {

    anvil::RK4<double> solver;

    std::cout
        << "Anvil OK: "
        << solver.name()
        << std::endl;

    return 0;
}

Notes

Note: Anvil is optimized for performance-critical numerical workloads.

Tip: Use compiler optimizations (-O2 or -O3) for best performance.
