A modern C++ library for numerical simulation of differential equations

Anvil is an open-source C++ library for solving ordinary differential equations (ODEs) and building high-performance numerical simulations for scientific and engineering systems.

Overview

Anvil is built around a layered architecture that separates mathematical models, numerical solvers, and computational backends.

This design allows users to define differential systems independently from the numerical method used to solve them, enabling flexibility and extensibility.

The library uses modern C++ features such as templates, compile-time polymorphism, and zero-cost abstractions to ensure performance comparable to hand-written numerical code.

Scope

Anvil focuses on numerical simulation of ordinary differential equations (ODEs) and dynamical systems.

It provides tools for:

  • Time integration of dynamical systems
  • Explicit and adaptive numerical solvers
  • State vector evolution
  • Basic linear algebra operations

It does not aim to replace general-purpose symbolic mathematics libraries.

Key features

  • Header-only design (no build step required)
  • Modern C++17 implementation
  • Explicit and adaptive solvers
  • Extensible solver architecture
  • High-performance numerical execution

Applications

Anvil is suitable for scientific computing and engineering domains where continuous-time simulation is required.

  • Physics simulations
  • Control systems
  • Mechanical modeling
  • Electrical circuit simulation
  • Applied mathematics research

The library is designed for reproducibility, stability, and numerical accuracy in time-dependent systems.
